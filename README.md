# Practice system

## 实现功能

用户功能，登录、注册、查询数据库

------

## 项目部署

### 数据库

数据库表中类型都是为 varchar，建了简单表， user表结构如下：

username|password|email
---|:--:|---:
user|user|user@163.com
admin|admin|admin@163.com

### 项目初始化
```
npm install
可能会出现缺少modules错误，把node_modules删除重新cnpm install
```

### 后端部署和运行
```
cd service
npm install
node app.js
```

### 运行前端项目
```
npm run serve
```
